# Disaster Management System

## Overview

The Disaster Management System is a web-based platform designed to provide disaster-related information, emergency services, and awareness resources. The system helps users access important disaster management services, view live maps, and obtain guidance during emergency situations through an interactive and user-friendly interface.

## Features

* User-friendly and responsive interface
* Disaster awareness and information portal
* Emergency service information
* Live map integration
* Contact and support pages
* Service management modules
* Interactive web pages using JavaScript

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQL Database

## Installation

1. Clone the repository

```bash
git clone https://github.com/23211A66D0/disaster-management-system.git
```

2. Navigate to the project directory

```bash
cd disaster-management-system
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
node app.js
```
